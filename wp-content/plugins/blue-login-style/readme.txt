== Description ==

A tiny plugin which allows to customize your wordpress login page with beautiful styles. be happy :)

= Features =

*   Ability to Choose Beautiful Styles In `Blue Login Settings` Panel.
*   Ability to add message in login page.
*   Change login logo as piece of cake.
*   Ability to set login&logout&register redirect links.
*   And so on...

== Installation ==

1. Upload `blue-login` files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Select your favorite style in `Blue Login Settings` panel.
4. Set your custom logo in `Select Login Logo` panel.
